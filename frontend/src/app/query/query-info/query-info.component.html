Adatok lekérése a játékszerverről

Az adatok a Clash Royale hivatalos API szerveréről történnek lekérdezésre.

Az API válaszai a játékosok adatai esetében 51 másodpercre gyorsítótárazásra kerülnek, míg a klánok adatai 47 másodpercre kerülnek gyorsítótárazásra (ezek az idők megegyeznek az API gyorsítótárazásával) a felesleges lekérések elkerülése, illetve a gyorsabb oldalletöltés érdekében.

Játékos adatainak lekérése
  • A játékos azonosítója 0289CGJLlPQRUVY karaktereket tartalmazhat, és 8 illetve 9 karakter közti hosszúságú lehet.
  • Az azonoítót # előtag nélkül kell megadni
Klán adatainak lekérése
  • A klán azonosítója 0289CGJLPQRUVY karaktereket tartalmazhat, és 6 illetve 8 karakter közti hosszúságú lehet.
  • Az azonoítót # előtag nélkül kell megadni
